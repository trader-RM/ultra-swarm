# Ultra Swarm

Integration base for OpenCode Swarm, ECC, OpenSkills, and Ralphy.

> This pass is setup only. No behavior has been implemented.

## Structure

- `base/` — Base code (OpenCode Swarm)
- `integrations/` — Integration layers (swarm, ecc, opencode-agent-skills, openskills, ralphy)
- `configs/` — Configuration files
- `scripts/` — Utility scripts
- `docs/` — Documentation
- `tests/` — Test suite
- `.opencode/` — OpenCode agent/skill/command/plugin definitions
- `tmp/` — Temporary working files (gitignored)
